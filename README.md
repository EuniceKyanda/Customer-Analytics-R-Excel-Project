## Customer Analytics Project (R & Excel)

## Objective
Analyze rider behavior to identify opportunities to convert casual users into registered subscribers and improve predictable revenue.

## Tools
- R
- Excel

## Approach
- Cleaned and explored dataset in R
- Compared casual vs registered usage patterns
- Built Excel visuals to communicate trends

## Key Insights
- Registered users showed higher frequency and consistency
- Casual riders peaked on weekends, indicating opportunity for targeted promotions

## Recommendation
Introduce weekday incentives and loyalty offers to convert casual riders into subscribers.
